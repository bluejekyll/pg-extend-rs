Creates a new Allocated type from Postgres.

This does not allocate, it associates the lifetime of the Allocator to this type. it protects the wrapped type from being dropped by Rust, and uses the associated Postgres Allocator for freeing the backing memory.

Safety

The memory referenced by ptr must have been allocated withinn the associated memory_context.
